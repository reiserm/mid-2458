========
midtools
========

`midtools` is a toolbox containing algorithms to analyze data acquired at `European XFEL <https://www.xfel.eu/>`_.

In preparation for experiment p2458 we will develop algorithms that process the data and store the results in HDF5-files.
Basic information are provided under **Getting Started**.
